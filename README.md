# Reclaim

A manifesto and a strategy for personal, free-to-run, data-sovereign software.
**No server. No account. No middleman. The only cost is tokens.**

**Live:** https://harryf.github.io/reclaim/

One self-contained `index.html` — runs anywhere static files are served, including
GitHub Pages, for free. The page is its own proof: a single file, no build step, no
backend, no tracking.
